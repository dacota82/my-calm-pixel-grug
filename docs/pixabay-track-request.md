# Pixabay Track Request

My Calm — Pixel Grug BGM용 Pixabay 음원 4곡이 필요합니다.

API 검색이 아니라, 주군이 Pixabay에서 직접 선곡·다운로드한
Local Audio Asset을 사용합니다.

### 명상음악 (2곡)

- calm piano meditation
- peaceful mindfulness

### 수면음악 (2곡)

- lullaby calm night
- deep sleep peaceful

## 필요한 파일 (4개)

| 구분 | 목표 파일명 | 저장 위치 |
|------|-------------|-----------|
| 명상 1 | `meditation-01.mp3` | `public/audio/pixabay/` |
| 명상 2 | `meditation-02.mp3` | `public/audio/pixabay/` |
| 수면 1 | `sleep-01.mp3` | `public/audio/pixabay/` |

## Source Page URL 기록 방법

1. Pixabay에서 곡 페이지를 엽니다.
2. 브라우저 주소창의 URL 전체를 복사합니다.
3. 예: `https://pixabay.com/music/...`
4. 곡마다 1개의 Source Page URL을 한신에게 전달합니다.

## Creator 기록 방법

1. Pixabay 곡 페이지의 업로더/아티스트 이름을 확인합니다.
2. 표시된 Creator 이름을 그대로 기록합니다.
3. 곡마다 Creator를 함께 전달합니다.

## License 확인 방법

1. Pixabay Content License를 확인합니다.
2. 앱에서는 `Pixabay Content License`로 표기합니다.
3. 독립 Audio Library처럼 재배포하지 않고, My Calm BGM으로만 사용합니다.

## 전달해 주실 정보 체크리스트

각 곡마다 다음을 알려 주십시오.

- [ ] 목표 파일명 (`meditation-01.mp3` 등)
- [ ] 실제 파일 (`public/audio/pixabay/`에 저장)
- [ ] 표시용 Title
- [ ] Creator
- [ ] Pixabay Source Page URL
- [ ] License 확인 완료

## 금지 사항

- Pixabay CDN/페이지 Audio URL Hotlink 금지
- 가짜 MP3 / 다른 사이트 음원 대체 금지
- 앱 내 MP3 다운로드 버튼 금지
