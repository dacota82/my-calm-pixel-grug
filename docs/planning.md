My Calm — Pixel Grug는 사용자가 하루 중 짧은 순간에 자신의 마음을 알아차리고, 억지로 바꾸거나 해석하기보다 **작은 픽셀 카드로 남기도록** 돕는 감정 체크인 앱이다.

이 앱은 명상을 가르치는 앱이 아니라, **오늘의 마음을 조용히 바라보는 작은 의식**이다.

> 하늘은 구름을 붙잡지 않는다.  
> My Calm도 감정을 고치려 하지 않는다.  
> 그저 오늘의 마음을 보고, Grug와 함께 작은 카드 하나로 흘려보낸다.

---

## 2. 기획 배경

Calm 같은 기존 명상·수면 앱은 명상, 수면, 호흡, 음악, Daily, Sleep Stories, 기업복지, 헬스케어까지 확장한 **대형 웰니스 플랫폼**이다.

이미 큰 콘텐츠 플랫폼이 있는 상황에서, 같은 방식으로 더 많은 명상·수면 오디오를 제공하는 것은 차별성이 약하다.

**My Calm은 Calm을 따라가지 않는다.**  
더 많은 콘텐츠가 아니라, 짧은 순간에 마음을 바라보고, 숨을 한 번 고르고, 작은 픽셀 카드로 남기는 앱을 지향한다.

기획의 출발점은 **새벽 산행에서의 알아차림**이다. 같은 산길을 걸어도 매일의 하늘·바람·마음은 다르다. 이 앱은 그 다름을 판단하거나 고치지 않고, 조용히 바라본 뒤 작은 기록으로 남긴다.

---

## 3. 문제 정의

많은 사용자는 마음을 돌보고 싶지만, 기존 웰니스 앱을 꾸준히 쓰기 어렵다.

- 긴 명상 콘텐츠는 부담스럽다.
- 긴 일기 작성은 귀찮다.
- 앱을 열었을 때 무엇을 해야 할지 모를 때가 많다.
- 지금 느낀 감정은 시간이 지나면 쉽게 사라진다.
- 기존 앱은 좋은 콘텐츠를 주지만, **지금 이 순간을 짧게 기록**하는 경험은 상대적으로 약하다.

→ 필요한 것은 더 많은 콘텐츠가 아니라, **짧고 가벼운 감정 체크인 경험**이다.

## 4. 핵심 콘셉트

> Calm이 좋은 콘텐츠를 오래 듣는 앱이라면,  
> My Calm은 지금 내 마음을 짧게 바라보고  
> 작은 픽셀 카드 하나로 남기는 앱이다.

- 감정을 분석·해결하도록 강요하지 않는다.
- 지금의 마음을 선택하고, 짧게 숨을 고르고, Grug의 한마디와 함께 픽셀 카드로 남긴다.
- 아무 때나 오래 쓰는 앱이 아니라, **마음이 흔들리는 짧은 순간**에 쓰는 앱이다.

## 5. 타겟 사용자

### 5.1 1차 타겟
- 긴 명상 앱이 부담스러운 사용자
- 긴 일기보다 짧은 감정 기록을 원하는 사용자
- 하루 중 짧은 회복 루틴이 필요한 사용자
- 픽셀 아트·캐릭터 감성을 좋아하는 사용자
- 오늘 내 마음이 어땠는지 가볍게 남기고 싶은 사용자

### 5.2 상황 기반 타겟

| 사용 상황 | 사용자 니즈 | My Calm의 대응 |
|-----------|-------------|----------------|
| 새벽/아침 | 오늘 마음을 확인하고 싶다 | 오늘의 마음 묻기 |
| 점심 이후 | 집중력이 흐트러진다 | 짧은 호흡과 회복 |
| 낮잠 전후 | 잠깐 쉬고 다시 시작하고 싶다 | 쉬어도 된다는 안내 |
| 퇴근 후 | 하루 감정을 내려놓고 싶다 | 오늘 감정 카드 기록 |
| 잠들기 전 | 생각을 줄이고 싶다 | 조용한 마무리 체크인 |

## 6. 핵심 가치 제안

**오늘의 마음을 Grug와 함께 1분 안에 정리하고, 픽셀 카드로 남긴다.**

| 가치 | 설명 |
|------|------|
| 짧다 | 1분 안에 끝나는 감정 체크인 |
| 가볍다 | 긴 명상·긴 일기 없이 감정 선택과 한 줄 입력만 |
| 남는다 | 오늘의 감정이 픽셀 카드로 기록·누적됨 |

## 7. Calm과의 차별화

| 구분 | Calm | My Calm — Pixel Grug |
|------|------|----------------------|
| 본질 | 수면·명상 콘텐츠 플랫폼 | 1분 감정 체크인 의식 |
| 사용 방식 | 콘텐츠를 선택해 듣고 소비 | 현재 감정을 선택하고 기록 |
| 핵심 자산 | Sleep Stories, 음악, 전문가 콘텐츠, 브랜드 | Grug 캐릭터, 픽셀 카드, 감정 기록 |
| 고객 관계 | Daily 콘텐츠와 구독 기반 루틴 | Grug 체크인과 카드 누적 |
| 수익 구조 | 개인 구독, B2B, 헬스케어, 파트너십 | 초기 무료 웹 MVP, 향후 캐릭터팩·테마팩 |
| 초기 방향 | 대형 콘텐츠 도서관 | 작고 반복 가능한 1분 루틴 |

My Calm은 Calm을 **대체하려 하지 않는다**.  
Calm이 큰 산이라면, My Calm은 새벽 산길에서 잠깐 하늘을 보는 경험을 앱으로 옮긴다.

## 8. 주요 기능

### 8.1 시간대 기반 체크인

| 시간대 | 예시 문구 |
|--------|-----------|
| 아침 | 오늘 마음은 어떤가요? |
| 점심 이후 | 잠깐 숨을 고를까요? |
| 낮잠 시간 | 짧게 쉬어도 괜찮아요. |
| 저녁 | 오늘 하루를 내려놓아 볼까요? |
| 밤 | 잠들기 전 마음을 가볍게 해볼까요? |

### 8.2 감정 선택
평온, 지침, 불안, 감사, 집중, 회복, 시작

### 8.3 짧은 호흡
1. 숨 들이쉬기  
2. 잠시 멈추기  
3. 천천히 내쉬기  

초기 MVP: 복잡한 애니메이션 없이 **텍스트 + 간단한 타이머**

### 8.4 Grug 체크인
선택한 감정·시간대에 맞춰 짧은 문장 표시. 예:
- 큰 돌 말고 작은 돌 하나만.
- Grug도 오늘 조금 느림.
- 쉬는 것도 동굴 친구의 일.
- 오늘 돌 많이 들었음. 이제 내려놔도 됨.
- 생각 많음. 숨 하나 먼저.

### 8.5 픽셀 카드 기록
카드 포함 정보: 날짜, 시간대, 선택 감정, 사용자 한 줄, Grug 메시지, `assets/anchor-grug.png`

## 9. MVP 범위

### 9.1 반드시 할 것
- [ ] 시간대 자동 구분
- [ ] 감정 선택
- [ ] 짧은 호흡 안내
- [ ] Grug 체크인 메시지 표시
- [ ] 픽셀 카드 생성
- [ ] 카드 기록 저장 (`localStorage`)
- [ ] 카드 목록 표시

### 9.2 하지 않을 것
- 긴 명상 콘텐츠 / Sleep Stories / 오디오 / 셀럽 콘텐츠
- 복잡한 구독 / 초기 B2B·Health
- 로그인 / 결제
- 실시간 이미지 생성 API / 감정별 이미지 대량 생성

### 9.3 나중에 할 것
- 감정별 Grug 이미지, 카드 다운로드, 월간 감정 리포트
- Gemini Nano Banana API, 모바일 최적화, 위젯, 캐릭터팩·테마팩

## 12. 기대 효과

- 마음을 돌보는 **진입장벽을 낮춘다** (긴 명상·일기 불필요).
- Grug를 통해 앱을 **작은 동굴 친구**처럼 느끼게 한다.
- 픽셀 카드로 하루·감정 변화를 **가볍게 돌아볼 수 있게** 한다.

---

## 13. 최종 결론

My Calm — Pixel Grug는 Calm 같은 대형 콘텐츠 플랫폼을 지향하지 않는다.

새벽 산행처럼 조용히 현재의 마음을 바라보고,  
Grug와 함께 **1분 안에 감정을 기록**하는 앱이다.

초기 MVP는 **짧은 호흡 + Grug 체크인 + 픽셀 카드 기록**에 집중한다.
