# Pixel Grug

픽셀 아트 감성의 웹 앱 프로젝트입니다.

## 상태
- 새 폴더 + Git 저장소 생성 완료
- 시각 정체성 초안 완료
- 1차 기준 이미지(anchor) 확정
- SDD 문서(planning / technical-spec / design) 작성
- My Calm — Pixel Grug MVP 구현 (로컬)
